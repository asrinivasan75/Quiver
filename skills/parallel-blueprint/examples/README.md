# Examples

Reference outputs that demonstrate the target format and tone of a `parallel-blueprint` plan. The skill reads these to calibrate the level of detail expected in a generated blueprint.

## Files

- **`cubist-hackathon.pdf`** — a 24-hour hackathon proposal demonstrating the canonical pattern: Hour 0–1 interface freeze; 5 owners with one directory each; mocked contracts everywhere. This is the reference example cited in `SKILL.md`.

## Adding a new example

A good example shows the full 12-section blueprint, a non-trivial workstream allocation table (3+ owners), and concrete frozen contracts (real schemas/signatures, not pseudocode). Drop the PDF in this directory and reference it from `SKILL.md`'s "Reference example" section.
